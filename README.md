💸 AI UPI Transaction Analyzer

Making Sense of Your Daily Transactions
🚀 What is this project?

This project is a simple financial analysis tool that helps you understand your UPI transactions better.

Instead of just looking at a long list of payments, it turns your transaction data into:

Clear summaries
Visual insights
Basic spending analysis

The goal is straightforward:
👉 help you see where your money is going and how you can manage it better

🔍 Why I built this

UPI apps show transactions — but they don’t really help you understand your spending.

I built this project to:

Work with messy real-world transaction data
Extract useful information from it
Present it in a way that’s actually easy to understand
⚙️ How it works
Transaction CSV
     ↓
Data Cleaning (Regex + Pandas)
     ↓
Structured Data
     ↓
Analysis (Income, Expenses, Balance)
     ↓
Visualization (Charts + Tables)
     ↓
Insights (Rule-based suggestions)
📊 What it shows
Total income and total spending
Net balance
Number of transactions
Spending distribution (via charts)
A full list of your transactions
🧠 Insights it provides

The app also gives simple observations like:

Your highest spending category
How much you spend on things like food delivery
Suggestions to improve your spending habits

These are based on rules, not actual AI (yet).

🛠️ Tech used
Python
Pandas
Streamlit
Plotly
Regex
▶️ How to run it
git clone https://github.com/your-username/ai-upi-analyzer.git
cd ai-upi-analyzer
pip install -r requirements.txt
streamlit run app.py
⚠️ Things to know
This is not connected to any bank or live data
It works with a CSV file (sample provided)
Insights are rule-based, not AI-generated
Categories are not fully automated yet
